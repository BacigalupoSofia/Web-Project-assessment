# Fencing School Vina del Mar Website

A static website for the EEVM fencing school in Viña del Mar. The site includes a home page and supporting pages for membership information, timetable details, and contact information.

## Project Structure

- `index.html` - Home page for the fencing school
- `pages/membership.html` - Membership details and enrollment information
- `pages/timetable.html` - Practice schedule and class times
- `pages/contact.html` - Contact form and club contact information
- `stylesheets/stylesheet.css` - Global styles for the site
- `stylesheets/membership.css` - Styles specific to the membership page
- `stylesheets/timetable.css` - Styles specific to the timetable page
- `stylesheets/contact.css` - Styles specific to the contact page
- `images/` - Site images, including logo and fencing graphics

## How to Use

1. Open `index.html` in a web browser.
2. Use the navigation links to visit Membership, Timetable, and Contact pages.

## Notes

- The site is built with HTML and CSS only.
- No server or build tools are required.
- Ensure the `images/` and `stylesheets/` folders remain in place relative to the HTML files.
